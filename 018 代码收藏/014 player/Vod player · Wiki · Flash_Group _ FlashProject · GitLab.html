Web播放器接口

1.播放器入口协议

本文档定义了web播放器的接口规范和接口详细定义。

  • flash地址播放模式
  • flash地址规则
接口协议
http://player.pptv.com/flashplayer/[k1]/[v1]/[k2]/[v2]/.../[kn]/[vn].swf
  • flashvars接口参数
参数名 是否必填 描述
pid 必选 aplusCMS里的播放器渠道id
vtype 选填 vtype=0 启动flv播放器 （启动flv播放器必填）
flv 选填 启动flv播放器时，视频地址 （启动flv播放器必填）
version 选填 flv播放器版本的版本号 （启动flv播放器必填）
api 选填 为flash回调js方法的挂载对象
asc 选填 播放器是否允许单双击（all/click/doubleclick/none）,默认值为 all
act 选填 播放器是否跟JS交互，默认值为always
title 选填 频道名称
link 选填 pptv网站对应的播放地址
swf 选填 分享的swf地址
clid 选填 分类id
type 选填 用户端类型,web端播放器类型为'web.fpp'
dot 选填 内容打点工具
smart 选填 smartclick打点工具
playStr 选填 pp云播放串
priplay 选填 内部直接使用的play接口，无需加工
aplus 选填 外部传入的aplus地址
goHomeType 选填 去首页跳转形式，默认值为’_top‘
vw 选填 播放器的宽度
vh 选填 播放器的高度
o 选填 渠道号
buskin 选填 播放器备份皮肤url
ctx 选填 参数上下文（key1={p/d/a/v}value1&key2={p/d/a/v}value2&…大括号中的字符代表了透传方向，p—play；d—dac；a— ads；v—vod，透传方向前缀可以组合）
advars 选填 透传给广告的参数
stime 选填 视频开始播放时间
etime 选填 视频结束播放时间
isdanmu 选填 是否支持弹幕 0~1为关闭弹幕功能 2为开启弹幕,但默认不显示 3为开启弹幕,默认显示
user_name 选填 ppyun cp帐号
danmutk 选填 ppyun 弹幕token(根据passport方式加密)
pay 选填 是否是付费影片 1是付费 0是免费
fimg 选填 视频封面图截图地址
rp 选填 直播回看是否轮播
ap 选填 是否自动播放（false表示初始暂停）
hl 选填 hide logo 隐藏品牌logo
ph 选填 play服务host

2.播放器可调用接口 （js&as）

Web播放器提供给外部的调用方法。

Js部分通过ExternalInterface.addCallback(name,fun);

As 部分直接调用接口

1) 视频播放

方法名称 用途 参数类型
playVideo 视频播放调用(处于暂停或停止播放时恢复当前视频的播放) Object
  • 参数格式
参数名 是否必填 描述
pl 选填 播放串
cid 选填 播放频道id (和pl参数必须传一个)
title 选填 频道名称
link 选填 pptv网站对应的播放地址
swf 选填 分享的swf地址
playStr 选填 pp云播放串

2) 视频暂停

方法名称 用途 参数类型
pauseVideo 视频暂停

3) 视频停止

方法名称 用途 参数类型
stopVideo 视频停止

4) 拖动视频

方法名称 用途 参数类型
seekVideo 拖动视频 int

5) 切换码流

方法名称 用途 参数类型
switchVideo 切换码流 int

6) 设置播放器音量

方法名称 用途 参数类型
setVolume 设置播放器音量 int 0-100

7) 获取播放器的音量

方法名称 用途 参数类型 返回类型
getVolume 获取播放器的音量 int (0--100)

8) 获取当前视频截图

方法名称 用途 参数类型 返回类型
getSnapshot 获取当前视频截图 BitmapData

9) 获取当前播放器版本

方法名称 用途 参数类型 返回类型
getVersion 获取当前播放器版本 String

10) 设置播放器尺寸

方法名称 用途 参数类型
setSize 设置播放器尺寸 Number
  • 参数格式
参数名 是否必填 描述
w 必填 播放器宽度
h 必填 播放器高度

11) 切换静音

方法名称 用途 参数类型
setMute 切换静音 Boolean

12) 当前播放点

方法名称 用途 参数类型 返回类型
playTime 当前播放点 Number

13) 视频时长

方法名称 用途 参数类型 返回类型
duration 视频时长 Number

14) 接口实现外部设置用户行为功能

方法名称 用途 参数类型
setBfrRecord 接口实现外部设置用户行为功能,会跟bfr报文一起发送 String

15) 得到播放器的播放状态

方法名称 用途 参数类型 返回类型
playState 得到播放器的播放状态 String（5-播放/6-暂停/8-自动播放结束）

16) 外部传递querystring

方法名称 用途 参数类型
setCtx 外部传递querystring String

17) JS设置配置中心数据

方法名称 用途 参数类型
setConfig JS设置配置中心数据 Object

18) 外部 javascript 已经检测到PPAP

方法名称 用途 参数类型
hasPPAP 外部 javascript 已经检测到PPAP

19) 播放器通用接口

方法名称 用途 参数类型
setCallback 播放器通用接口 Object
  • 参数格式

{ "header": 

{ "type"://type }

,

"body": 

 { "data"://obj}

}

  • type对照表
type 描述
userinfo 用于设置 用户信息及会员信息
theatre 剧场模式相关
barragesetting 设置弹幕相关
sendbarrage 发送弹幕
  • 参数格式

type = "userinfo"

参数名 是否必填 描述
PPKey 选填 passport返回的PPKey，登录用户必传
PPName 选填 passport返回的PPName，登录用户必传
ppToken 选填 passport返回的ppToken，登录用户必传
UDI 选填 passport返回的UDI，登录用户必传
UDR 选填 passport返回的UDR，登录用户必传
ids 选填 webcfg中的配置，如没有，则不传
tgs 选填 webcfg中的配置，如没有，则不传
source 选填 Flash内部点击弹出登录窗口的来源 如 切换低延迟直播时，为‘rtmp’ 具体参看 onVipValidate接口说明

type = "theatre"

参数名 是否必填 描述
mode 必选 1/0 打开/关闭剧场模式

type = "barragesetting"

参数名 是否必填 描述
visible 选填 弹幕输入框是否可见
alpha 选填 弹幕的透明度
color 选填 弹幕字体的颜色
layer 选填 弹幕的可见的层数
xbenable 选填 弹幕是否1开启/0屏蔽小冰

type = "sendbarrage"

参数名 是否必填 描述
color 选填 弹幕字体的颜色
content 必填 发送弹幕的内容

20) SA采集视频播放信息

方法名称 用途 参数类型
getPlayInfoDeliver 视频启动时提供给页面的SA采集视频播放信息

21)获取视频信息是否可以分享

方法名称 用途 参数类型
getShareDisable 获取视频信息是否可以分享

3.播放器事件

1) 消息机制

方法名称 用途 参数类型 返回类型
onNotification 消息机制（only js） object
  • 返回格式

{ "header": 

{ "type"://type }

,

"body": 

 { "data"://obj}

}

  • type对照表
type 描述
onoffbarrage display:true/false 弹幕是否开启
theatre mode:1/0 打开/关闭剧场模式
recomclick 点击后推荐
setupbarrage mode:1/0 弹幕初始化是否成功
nextvideo 视频播放结束时调用，用来js调用playVideo播放下一集。
playbarrage 弹幕数据
openSportVip 弹开通体育会员浮框（主要是购买各种类型的体育会员包）
buyVideo 弹购买单片浮框
usecoupon 弹购买观看券或者观赛券浮框

2) 播放器加载完成

方法名称 用途 参数类型 返回类型
onInit 播放器加载完成

3) 播放器加载完成

方法名称 用途 参数类型 返回类型
onReady 播放器初始化完成

4) 视频信息加载完成

方法名称 用途 参数类型 返回类型
onVideoReady 视频信息加载完成 Object
  • 返回格式
参数名 描述
duration 视频时长
totalbytes 总字节数
width 视频分辨率宽
height 视频分辨率高

5) 播放状态发生变化

方法名称 用途 参数类型 返回类型
onPlayStateChanged 播放状态发生变化 string
  • state对照表
aid 描述
1 init加载完成
2 ready初始化完毕
3 start开始
4 buffering缓冲
5 playing播放
6 paused暂停
7 stopped用户手动停止
8 ended自动播放结束

6) 播放模式发生切换

方法名称 用途 参数类型 返回类型
onModeChanged 播放模式发生切换 object
  • 返回格式
参数名 描述
3 全屏模式
4 退出全屏模式

7) 音量设置发生变化

方法名称 用途 参数类型 返回类型
onVolumeChanged 音量设置发生变化 object
  • 返回格式
参数名 描述
event 事件名称：volume音量大小改变，mute静音
value 退出全屏模式

8) 播放或者加载进度发生变化

方法名称 用途 参数类型 返回类型
onProgressChanged 播放或者加载进度发生变化 object
  • 返回格式
参数名 描述
bytesloaded 已加载的字节数
timeloaded 已播放的时间

9) 出现播放错误

方法名称 用途 参数类型 返回类型
onError 出现播放错误 object
  • 返回格式
参数名 描述
code 错误码
message 错误描述

10) 视频流改变后出发(如切码流)

方法名称 用途 参数类型 返回类型
onStreamChanged 视频流改变后出发(如切码流) object
  • 返回格式
参数名 描述
ft 切换序列 //0
rid 资源ID
bitrate 对应码流

11) 广告开始、结束、显示或者关闭

方法名称 用途 参数类型 返回类型
onAdEvent 广告开始、结束、显示或者关闭 object
  • 返回格式
参数名 描述
event 广告事件
aid 广告位ID
atid 广告ID

12) 会员特权事件

方法名称 用途 参数类型 返回类型
onVipValidate 会员特权事件 string
  • 返回格式
参数名 描述
3 点击vip按钮触发
5 点击登录按钮触发
web_newguanggao 会员免广告触发
web_web_new1080p 蓝光开会员触发
web_web_newjiasu 加速免广告触发
